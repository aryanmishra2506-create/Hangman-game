# Hangman-game
A simple text-based Hangman game in python
## How to Run
1. Clone this repository or download the file.
2. Open terminal/command prompt.
3. Run: `python hangman.py`
## Features
- Word guessing
- Limited attempts
- Console-based interaction
## Requirements
- Python 3.x installed on your system
